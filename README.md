# D365 Agent Action Explainability & Trust Dashboard
**Feature 1 Prototype** — Dynamics 365 CRM Principal PM Feature Analysis

## What this is
A React prototype of the Agent Action Explainability & Trust Dashboard proposed for Microsoft Dynamics 365 Sales. Designed for senior stakeholder review.

## Run locally
```bash
cd d365-agent-trust
npm install
npm run dev
```

## Views
- `/lead` — Lead Detail with Agent Activity Panel (main feature)
- `/manager` — Manager Aggregate View
- `/settings` — Agent Configuration (stub)

## Stack
React 18 · Vite · Tailwind CSS · lucide-react · recharts · React Router v6
